# Comfyui-IV2Z
修改[ComfyUI-VideoHelperSuite](https://github.com/Kosinkadink/ComfyUI-VideoHelperSuite)的VideoCombine节点，使其将编码后的视频变为压缩包进行输出

![image](https://github.com/ilslli/Comfyui-IV2Z/blob/main/image/node.png)

# 功能：
其他功能与原节点一致

（启用顺序由上到下，当上一个开关开启后下面的开关才会起效，单独开启开关不会起效）

save_as_zip：将最后的文件输出为zip压缩包格式

polyglot_png：组合图片与压缩包，输出的图片里将后缀改为zip可顺利解压

use_solid_shell：将输出图片变为纯色，内容不变
